¿que significa CSS?

CSS es el acronimo de "Cascading Style Sheets"

¿para que sirve CSS?

el CSS sirve para darle estilo y proporcinar visualmente el documento, para interpretarlo se usa el programa html

¿a que se llama regla CSS?

la regla predeterminada o la regla CSS es la estructura que se ultiliza al querer decorar un sitio web con CSS

¿cual es la anatomia de la regla CSS?

¿que es un selector en CSS?

un selector en CSS sirve para selecciones mas especificas. como selector de elemento,identificacion,clase,atributo y pseudoclase

indentificar selectores

selector universal:
para aplicar estilos a todos los elementos html de un documento
selector de clases:
sirve para que los clientes web lo identifiquen y Lo distingan del resto
selector de ID:
es igual al selector de clase solo que se aplica a un unico elemento html
selector por etiquetas:
para aplicarle propiedades CSS a una etiqueta
selector descendente:
para seleccionar un elemento que esta dentro de otro del mismo tipo

    1: el navegador carga el html 2: convierte el html en dom,el dom representa el documento en la memoria del ordenador 3:Entonces, el navegador va a buscar la mayor parte de los recursos vinculados al documento HTML, como las imágenes y los videos incrustados... ¡y también el CSS vinculado! 4: el navegador analiza el CSS y ordena en diferentes "cubos" las diferentes reglas segun el tipo de selector 5: el arbol de renderizacion presenta la estructura en que los nodos deben aparecer despues de aplicarle las reglas
